# 基于 Gin 与 Nuxt.js 的跨境电商独立站 - 项目经历

## 一、项目概述

**项目名称**：跨境电商独立站设计与实现  
**项目周期**：2025 年（毕业设计项目）  
**项目角色**：全栈开发 / 项目负责人  
**技术栈**：Go + Gin + Vue3 + Nuxt 4 + MySQL + Redis + PayPal  

本项目是一套面向跨境电商场景的完整独立站解决方案，涵盖管理后台、前台商城、后端 API 三大子系统。支持商品管理、订单交易、PayPal 跨境支付、多端认证、内容管理等功能，采用前后端分离与分层架构，具有良好的可扩展性与可维护性。

---

## 二、项目职责与贡献

### 2.1 后端架构设计与实现

- **分层架构设计**：采用 Handler → Service → Repository 三层架构，通过 `RepositoryFactory` 与 `ServiceFactory` 统一管理依赖注入，实现业务逻辑与数据访问解耦，便于单元测试与模块扩展
- **API 路由规划**：将接口划分为公开路由、管理端（`/api/manage`）、客户端（`/api/client`）三大组，分别对接不同认证策略，支持多端复用同一数据层
- **数据库建模**：基于 GORM 设计 30+ 张业务表，涵盖商品（含 SKU、属性、属性值）、订单、用户、CMS、支付日志等核心领域，合理使用索引与关联关系
- **双认证体系**：管理端采用 Redis Session 认证（支持 Header / Query / Cookie 多种 Token 来源）；前台采用 JWT + 设备指纹（`DeviceFingerprintMiddleware`），支持游客与登录用户差异化访问
- **PayPal 支付集成**：接入 PayPal Sandbox API，实现创建订单、捕获支付、退款全流程；支持支付成功邮件通知（gomail）；预留 Webhook 接口以支持异步支付回调

### 2.2 管理后台开发

- **技术选型**：基于 Vue 3 + Vite + Ant Design Vue 4.x + Pinia 构建 SPA 管理后台，使用 TypeScript 保证类型安全
- **权限控制**：实现 RBAC 权限模型（管理员-部门-角色-权限），配合 Vue 自定义指令 `v-permission` 实现按钮级权限控制；路由守卫统一校验 Token 与用户信息
- **核心业务模块**：
  - **商品管理**：支持多规格 SKU 配置、属性/属性值管理、类目树、标签关联、富文本详情（WangEditor）、图片裁剪上传
  - **订单管理**：订单列表、状态流转、发货录入、退款发起、PayPal 退款对接
  - **内容管理**：文档/文章 CRUD、推荐位与推荐项、标签管理
  - **站点管理**：商店设置、站点信息、市场配置
  - **系统管理**：管理员、部门、角色、权限、前台用户管理
- **数据可视化**：仪表板展示总用户数、商品数、订单数、销售额等核心指标；使用 ECharts 绘制用户增长趋势与订单增长趋势图

### 2.3 前台商城开发

- **技术选型**：使用 Nuxt 4 + Nuxt UI 4 + Tailwind CSS 构建 SSR 商城前端，支持 SEO 优化与预渲染
- **购物流程**：实现首页、分类/集合、商品详情、购物车、结算、订单详情完整链路；购物车采用 `useCartShared` composable 实现多页面状态共享与数量同步
- **用户体系**：支持注册、登录（JWT）、收货地址管理；支持游客下单（仅邮箱）与登录用户下单
- **支付流程**：集成 PayPal 支付，支持跳转至 PayPal 授权页、支付成功后回调更新订单状态并发送邮件
- **国际化能力**：前台表单验证、错误提示、文案结构支持多语言扩展（当前以英文为主）

### 2.4 配置与运维支持

- **配置管理**：使用 Viper 加载 YAML 配置，集中管理 MySQL、Redis、邮件（SMTP）、PayPal、前端 URL 等参数，便于环境切换
- **跨域与静态资源**：全局 CORS 中间件支持前后端分离部署；`/api/oss` 映射本地 OSS 目录，支持文件上传与静态资源访问
- **数据库脚本**：提供 SQL 初始化脚本，便于新环境部署与数据迁移

## 三、技术亮点

| 领域 | 实现方式 |
|------|----------|
| **前后端分离** | 管理端、前台、后端三端独立部署，通过 HTTP API 通信 |
| **双端认证** | 管理端 Redis Session；前台 JWT + 设备指纹，支持游客与登录用户 |
| **跨境支付** | 集成 PayPal Sandbox，支持创建订单、捕获、退款、Webhook 预留 |
| **状态共享** | Nuxt `useState` + `useAsyncData` 实现购物车跨页面状态共享 |
| **权限控制** | RBAC + 自定义指令，实现菜单与按钮级权限 |
| **分层架构** | Handler-Service-Repository + Factory 模式，便于测试与扩展 |
| **响应式设计** | 前台采用 Nuxt UI / Tailwind，适配多端访问 |
