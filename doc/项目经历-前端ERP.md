## 三、核心职责与成果

### 3.1 项目搭建与工程化

- 基于 Vite 搭建 Vue 3 + TypeScript 工程，配置路径别名、环境变量、开发代理等
- 按模块划分目录结构：`api`（接口）、`views`（页面）、`components`（组件）、`router`、`store`、`utils`、`directive`
- 统一封装 Axios 实例，实现请求拦截（Token 注入）、响应拦截（统一错误处理、登录态失效跳转）、超时与异常提示
- 集成 Ant Design Vue，结合 `theme.less` 做主题定制

### 3.2 通用组件封装

- **ApiBasicTable**：可复用的表格组件，支持 API 驱动的分页加载、搜索参数、滚动区域、插槽自定义列，在商品、订单、用户等 10+ 列表页复用
- **YexUpload 系列**：基于 cropperjs 封装图片裁剪上传（单图、多图、画廊），支持裁剪比例、预览，统一对接 OSS 接口
- **YexEditor**：集成 WangEditor，封装富文本编辑组件，用于商品详情、文档内容等
- **YexDict**：字典组件（DictSelect、DictLabelSelect），支持服务端枚举数据驱动下拉与标签展示
- **BasicModal**：标准弹窗封装，统一关闭、确认、加载态逻辑
- **PageLayout**：页面容器组件，统一卡片、间距、布局规范

### 3.3 布局与权限体系

- **Layout 布局**：左侧菜单 + 顶栏的经典后台布局，菜单支持折叠、响应式（移动端默认折叠），基于路由自动生成菜单
- **RBAC 权限**：与后端权限码联动，实现 `v-permission` 自定义指令，用于按钮/区域级权限控制；路由守卫中校验 Token，未登录跳转登录页
- **登录态管理**：Pinia `authStore` 管理 Token、用户信息、权限列表；Token 持久化至 localStorage，支持多来源（Header、Cookie）兼容

## 六、面试可延伸问题

1. **表格组件如何设计 API 驱动？**  
   通过 props 传入 `api` 函数和 `searchParam`，内部封装分页、请求、Loading，通过 `bodyCell` 插槽支持自定义列。

2. **权限指令如何实现？**  
   在 `mounted` 中读取用户 `permission` 数组，与 `binding.value` 比对，无权限时 `removeChild` 移除 DOM。

3. **如何做登录态失效处理？**  
   Axios 响应拦截器中判断 `code === 18000`，调用 `authStore.loginOut()`，并跳转 `/#/login`。

4. **多规格 SKU 表单如何联动？**  
   使用 `watch` 监听 `sku_list` 变化，开启 SKU 时自动取默认规格的价格、库存更新主表单字段。

5. **Vite 与 Vue CLI 对比？**  
   Vite 基于 ESM 开发，冷启动快；生产用 Rollup 打包，支持按需编译与更细粒度优化。
